# Anvilari – Commercial Plastic Bags Website

## Overview

This project is a responsive commercial product website developed for **Anvilari**, a plastic bags supplier focused on serving business clients with clear product information and an easy ordering experience. The goal was to provide a clear product catalog, simple ordering experience, and mobile‑friendly interface for business customers.

The application focuses on usability, performance, and real‑world interaction patterns such as form validation, loading states, and user feedback.
